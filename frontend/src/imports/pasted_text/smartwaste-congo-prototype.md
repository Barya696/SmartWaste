# SMARTWASTE PARKCACTIVE – REPUBLIC OF CONGO

Create a complete modern web application prototype called **SmartWaste ParkCactive** for the **Republic of Congo (Brazzaville)**.

The platform must be a professional Waste Management, Waste Traceability, Collection, Recycling, and Transformation Management System.

The system must support three different user roles with completely customized dashboards and permissions depending on how the user logs in.

---

## LOGIN SYSTEM

Create a modern login page where users choose one of three roles before authentication:

### 1. Citizen (Population)

Purpose:

* Report waste found in neighborhoods.
* Track reported waste status.
* Receive notifications when waste has been collected.
* Earn eco-points and environmental badges.

### 2. Collector (Waste Worker)

Purpose:

* Receive collection assignments.
* Locate reported waste using GPS.
* Manage containers and collection routes.
* Update waste collection status.

### 3. Administrator (PARKCACTIVE)

Purpose:

* Full supervision of waste lifecycle.
* Monitor collection operations.
* Track waste from report to recycling/transformation.
* Manage blockchain traceability.
* Manage users and organizations.
* Generate compliance and traceability reports.

The application must automatically display the correct interface according to the selected role.
